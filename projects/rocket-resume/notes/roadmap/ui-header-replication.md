# UI: Header replication

Order: 2
Priority: P1
Stage: Ready
Status: Done
Track: Pages & UI Shell

Quick replication of the legacy header so Phase 0 pages can ship with a recognisable shell — not a from-scratch redesign. Took the legacy structure and stripped what doesn't apply to the new app: removed the search bar, removed the currency switcher, changed the auth entry point to redirect (the legacy app opens a modal). Full header redesign happens later once the broader new design is signed off and the auth modals are in place.
