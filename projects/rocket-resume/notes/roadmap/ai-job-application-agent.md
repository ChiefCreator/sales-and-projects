# AI job-application agent

Priority: P3
Stage: Refine
Status: Not started
Track: Future Directions

An AI agent that acts on the user's behalf — fills application forms on external sites, sends outreach emails, scans job boards for matching roles, with user-permissioned autonomy levels (review-each-action vs. full auto).

Distinct from the AI resume reviewer (which only edits the resume document). Far-future; depends on tenant infra, auth, and a partner / data network.
